Setting a custom domain for GitHub pages with Dot.tk

1. Prepare

- Sign up for a Github account

- Register a domain name on the Dot.tk (Note: when searching for domain names do not enter ".tk" at the end)

2. Domain linked section

- Setting domain


  + First, you have to log in my.freenom.com

  + Next, click Manage Domain

 

  + Continue, click Manage Freenom DNS

 

  + Next, click Add Records

    In Target  enter IP 185.199.108.153, this is the github IP page

     Add Records

  •     In Name you enter WWW
  •     In Type you click and select CNAME
  •     In Target you enter the domain just register. Ex (tranminhquang.tk)
  •     Note, Save Changes after each addition

- Setting on Github


  + Create a New repository (Review the instructions in Create a static website with Github)

  + After you've created a new repository, click Setting

  + Drag the web page down to the section Github Pages

  + After reloading the page, drag the web page down to the section Github Pages

  + In the Custom domain box, enter the domain name you have registered and click Save

Wait for 5-10 minute, into the search box type your domain name and enjoy.
