Create a static website with Github

A static website is a website that only consists of HTML, CSS and Javascript. You can build a completely free static web for  learning or even for business purposes with https://github.com (eg 17130182.github.io). Here are the steps to do it:

 

1. Create an account and log in to Github

This step is very simple, similar to most other websites so I will not talk in detail.

2. Create a new repository

- Notice, the repository name must be of the form  username.github.io  (Where username is your username, for example the account is 17130182, I will name the repository as 17130182.github.io)

 

3. Clone repository and push website up

- After creating a new repository, we will clone to the device.

- You can use the command on the Terminal, but to ease it we can use the Git Desktop application by clicking Set up in Desktop. There will be a dialog box asking to open the Git Desktop Application (if your computer is not already installed, it will move to the settings page).

- After turning on Github Desktop a window will appear for you to select the folder containing the project, the application will clone the repository on Github into this folder. Then you just need to copy the entire source code into that folder and Commit and push it to Github. Note:The homepage of the website must be named index.html

- Just as simple as that, we have a completely free website. Now open up your browser, type in http://username.github.io/ (ví dụ http://17130182.github.io/) and enjoy the result <3

Attention

- In case the repository has already created then select Clone in Desktop

- Or if the web browser does not automatically turn on the Desktop App, we turn on the Desktop App, select Clone repository

- Then repeat the steps as guided above.

  + P/s: Refer to Custom domain user guide here: Setting a custom domain for GitHub pages with Dot.tk
